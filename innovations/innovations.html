Step Start

Another notable development is the Step Start. As the name suggests, the Step Start allows the power supply’s high voltage transformer to “power up” in stages. This is a device designed to provide a reduced voltage at start up, reducing inrush. Another benefit is that secondary voltage is also reduced, reducing the stress on diodes and Magnetron Tube. This in hand can improve the service life of tubes and diodes, which results in less down time and reduced maintenance costs.

Power Supply Control Retrofit

Thermo Wave has realized the need for a modern Microwave Power Supply control and has developed a set of PLC based Power Supply Controls that will replace and/or eliminate many of the obsolete, difficult to obtain and problematic components in most of the existing Power Supplies in the industry. These are:

XC-Standard

This is a Filament and Electromagnet Control device that can be used on any Raytheon, Amana, Ferrite or Amtek Transmitter. The XC-Standard will allow full control from a system level control via raise/lower signal, analog signal or networked communication. Additionally, this device includes an Electromagnet Closed Loop Control option for Automatic Power Control using the most advanced electromagnet and filament control algorithms in the industry. The XC-Standard is contained in a stand-alone “Plug In” box. Items replaced will include:

  • Rheostat / Powerstat
  • Electronic Filament Control (Rheostat, Spellman, Smart Glo, EX Glo, Watlo Controller or Eurotherm Controller)/ Powerstat
  • Electronic Filament Control / Electronic Magnet Control (DC Electromagnet Drive)
  • Eliminates Under Current Board
  • Eliminates Raise / Lower Board
  • Eliminates electromechanical timing relays

XC-Complete

This is a complete replacement power supply control. The XC-Complete includes all the functions of XC-Standard Filament and Electromagnet Control device and adds additional controls and interlocks for internal cooling and system interface. This device can be configured to emulate ANY Power Supply and can interface with ANY Control. The XC-Complete allows full local control as well as a selection of remote control options (raise/lower or analog signals or Ethernet or Modbus communications). In addition, interlock and power supply conditions can be communicated to a system or plant level control system for status annunciation. Items replaced include:

  • Rheostat / Powerstat
  • Electronic Filament Control / Powerstat
  • Electronic Filament Control / Electronic Magnet Control
  • Eliminates Under Current Board
  • Eliminates Raise / Lower Board
  • Eliminates nearly all electro-mechanical control relays & timers

Optional Touch Screen

Thermo Wave can supply an optional Touch Screen Control for each of the XC variants, mounted as required.

With all of its evolved features our circulators will undoubtedly ensure longevity and a cost efficient operation.
